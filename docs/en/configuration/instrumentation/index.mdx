# Configuring the Instrumentation

The Alauda build of OpenTelemetry Operator provides automatic instrumentation capabilities that enable you to collect telemetry data from your applications without modifying their source code. This powerful feature simplifies observability implementation by automatically injecting OpenTelemetry instrumentation libraries into your application containers.

This section covers the configuration of automatic instrumentation, including language-specific settings, SDK variables, exporter configuration, and advanced deployment scenarios.
