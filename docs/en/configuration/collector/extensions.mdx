# Extensions

Extensions provide auxiliary functionality to the Collector without directly processing telemetry data. They handle tasks such as authentication, health monitoring, and performance profiling, enhancing the Collector's operational capabilities.

## BearerTokenAuth Extension

The BearerTokenAuth Extension provides bearer token authentication for securing communication between the Collector and external systems. It manages token-based authentication credentials.

**Reference**: [BearerTokenAuth Extension Documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/extension/bearertokenauthextension/README.md)

## OAuth2Client Extension

The OAuth2Client Extension implements OAuth 2.0 client authentication for the Collector. It handles token acquisition and renewal for secure communication with OAuth-protected backends.

**Reference**: [OAuth2Client Extension Documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/extension/oauth2clientauthextension/README.md)

## File Storage Extension

The File Storage Extension provides persistent storage capabilities for Collector components. It's essential for components that need to maintain state across restarts, such as the Tail Sampling Processor.

**Reference**: [File Storage Extension Documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/extension/storage/filestorage/README.md)

## OIDC Auth Extension

The OIDC Auth Extension implements OpenID Connect authentication for securing Collector endpoints. It validates OIDC tokens and enforces authentication policies.

**Reference**: [OIDC Auth Extension Documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/extension/oidcauthextension/README.md)

## Jaeger Remote Sampling Extension

The Jaeger Remote Sampling Extension provides remote sampling configuration for Jaeger clients. It allows centralized management of sampling strategies across distributed applications.

**Reference**: [Jaeger Remote Sampling Extension Documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/extension/jaegerremotesampling/README.md)

## Performance Profiler Extension

The Performance Profiler Extension enables CPU and memory profiling of the Collector itself. It's useful for diagnosing performance issues and optimizing Collector resource usage.

**Reference**: [Performance Profiler Extension Documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/extension/pprofextension/README.md)

## Health Check Extension

The Health Check Extension exposes health check endpoints for monitoring Collector status. It provides liveness and readiness probes for Kubernetes deployments and load balancers.

**Reference**: [Health Check Extension Documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/extension/healthcheckextension/README.md)

## zPages Extension

The zPages Extension provides in-process web pages for debugging and monitoring the Collector. It offers real-time visibility into pipeline statistics, running configurations, and component status.

**Reference**: [zPages Extension Documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/extension/zpagesextension/README.md)
